# tamil_newsreader Tamil Voice TTS
A React-based web application that converts Tamil text, URLs, and PDFs into natural-sounding speech. Features include:
> Tamil Text-to-Speech: Converts written Tamil text into spoken audio.
> URL and PDF Support: Reads content directly from Tamil websites and PDF documents.
> Playback Controls: Seek forward and backward for easy navigation.
> Pitch Adjustment: Customize the voice pitch for a more personalized experience.
> Built with React: Leveraging the power and flexibility of React.js.
